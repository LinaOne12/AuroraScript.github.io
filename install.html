Installation Process

Download Rampage from Downloads and then extract it's components to its own folder. I recommend using Winrar for extracting files Winrar. after that open the Rampage folder and then put the "RampageFiles, and Rampage.asi" in your RDR2 or GTA V directory. You can read the readme if you would like for more info about Rampage. Make sure to also write permission to your "RampageFiles" folder before launching so you can save your settings, refer to the FAQ on how to do that.
